# Results

This folder contains small summary CSVs only. Raw predictions, checkpoints, prepared arrays, Grad-CAM image folders, and other large outputs are intentionally excluded from the public repository.

The included CSVs report the final results from the cleaned Colab workflow:

- `stanford_summary.csv`
- `thyroidxl_summary.csv`

Running the scripts may create additional CSV outputs in this folder. Heavy artifacts should remain in ignored folders such as `checkpoints/`, `outputs/`, or `gradcam_outputs/`.

Place the ThyroidXL dataset outside git, then pass its folder with --data_dir.

Expected structure:

/path/to/thyroidxl/
  train/
    train_annotations.json
    images/
      ...
    masks/
      ...
  test/
    test_annotations.json
    images/
      ...
    masks/
      ...

The code expects the annotation JSON files to contain an "info" dictionary keyed by patient ID.
Each patient entry should include:
  conclusion
  images
  nodule_1

Fields used from nodule_1:
  position
  width
  height
  depth
  TIRADS

Label mapping:
  conclusion 1 or 2 = malignant
  conclusion 3 or 4 = benign
